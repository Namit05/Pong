# Pong Game (Python)
Pong is a classic two-player arcade game where each player controls a paddle and tries to bounce a ball past the opponent.

It’s simple: don’t miss the ball.

Miss once, and your opponent scores.  
Miss too often… yeah, it gets frustrating real quick.

---

## Features of the Code

- Two-player paddle controls  
- Smooth ball movement  
- Ball collision with paddles and walls  
- Score tracking system  
- Increasing difficulty (ball speed)  
- Clean game loop using Turtle  

---

## Tech Used

- Python  
- `turtle` module for graphics  
- Game loop and collision logic  

---
## How to Play

- Right paddle: Up and Down arrow keys  
- Left paddle: W and S keys  
- Hit the ball using your paddle  
- Don’t let the ball pass you
- First to 5 points wins!

Stay focused — it speeds up fast.
